Procedure

In the User Interface (UI) provided for this experiment:
Instructions:-
1. You will get the random values in their respective random parameters Indicators.
2. According to the given data calculate Real part a’ of closed loop poles & Proportional gain Kc for each set of values.
3. Enter the answer calculated above in their respective control box.
4. Check the answer for every set of values by clicking on the check button provided at the right hand side.
5. If you are satisfied with the answers click on the submit button to log your answers in an Excel file.
